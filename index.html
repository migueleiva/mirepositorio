🌐 Mi Hosting Web en S3

Un laboratorio práctico para aprender a desplegar sitios en AWS

¿Qué es este laboratorio?

Este sitio fue creado como parte de un laboratorio donde aprendes a alojar contenido estático en Amazon S3, incluyendo cómo usar bucket policies, redirecciones, y más.

Características

  • ✅ Despliegue 100% en la nube
  • ✅ Accesible desde cualquier navegador
  • ✅ Costo ultra bajo (incluso gratuito)
  • ✅ Aprendizaje práctico sobre AWS

¿Cómo subirlo?

  1. Crea un bucket S3 público habilitado como "Sitio Web Estático"
  2. Sube los archivos de este repositorio
  3. Configura el index document como index.html
  4. ¡Accede a tu web desde el endpoint del bucket!
